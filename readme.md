A simple console game Brain Games.

## Installation

```
sudo npm install -g hexlet-project-sergeimakarovweb
```

## Usage

Start the game with the following commands:

Is even
```
brain-even
```
Calculator
```
brain-calc
```
Progression
```
brain-progression
```
Greatest common divisor
```
brain-gcd
```
Is Prime
```
brain-prime
```
